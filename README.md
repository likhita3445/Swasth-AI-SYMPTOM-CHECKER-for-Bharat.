# AI-SYMPTOM-CHECKER
The Smart Diagnosis System uses AI and NLP to analyze user symptoms through voice or text in multiple languages. It provides a simple risk level (low, medium, high) with basic health advice. Designed for rural areas, it works offline and helps users find nearby hospitals and emergency support.

**“It also provides emergency ambulance assistance, daily health report monitoring with smart reminders, support for healthcare schemes like Ayushman Card, and an easy doctor appointment system for complete end-to-end care.”**
 
# 🏥 AI Symptom Checker

### Smart Diagnosis System Using Artificial Intelligence

## 📌 Overview

This project is a **smart healthcare support system** that uses **Artificial Intelligence (AI)** to understand user symptoms and provide basic health guidance.

Users can **speak or type in regional languages** (Hindi, Telugu, Tamil, etc.), making it highly accessible for people in **rural and semi-urban areas**.

The system does not give a full diagnosis but provides a **risk level (Low / Medium / High)** and suggests next steps.

---
## 🚨 Problem Statement

* Limited access to doctors in villages and small towns
* Health apps mostly available only in English
* Lack of support for regional languages
* Poor internet connectivity in rural areas
* Complex interfaces not suitable for non-technical users

---
## 💡 Solution

This project provides a **simple, multilingual, and offline-friendly AI system** that:

* Understands symptoms in local languages
* Supports **voice and text input**
* Works even with **low or no internet**
* Provides quick health guidance
* Helps users find nearby hospitals

---
## ✨ Key Features

* 🌐 **Multilingual Support** – Works with regional languages
* 🎤 **Voice Input** – Easy for non-literate users
* 🤖 **AI-Based Analysis** – Symptom checking using NLP
* 📊 **Risk Level Output** – Low / Medium / High
* 📡 **Offline Mode** – Works without internet
* 🏥 **Nearby Hospital Finder**
* 🚑 **Emergency & Ambulance Support Info**

---
## 🔄 How It Works

1. User enters symptoms (voice or text)
2. Input is converted into text
3. AI processes the symptoms using NLP
4. System compares with database
5. Displays:

   * Possible condition
   * Risk level
   * Basic health advice
   * Nearby hospitals
